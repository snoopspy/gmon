Monitor and managed mode conversion tools
===

## Install
```
make install
```

## Uninstall
```
make uninstall
```

## gmon
```
syntax : gmon <interface> [<new interface>]
sample : gmon wlan0 mon0
```

## gman
```
syntax : gman <interface> [<new interface>]
sample : gman mon0 wlan0
```
